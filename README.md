vim-config
==========

Hugo's vim configuration, ready to deploy anywhere


installation
------------

To install (unix): 

    cd .vim
    git clone git://github.com/sanchon/vim-config.git .
    cp .vimrc ..
    git submodule init
    git submodule update



To install (windows)

    cd vim/vimfiles
    git clone git://github.com/sanchon/vim-config.git .
